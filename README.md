Данный репозиторий содержит Django-приложение, разрабатываемое в рамках курса по программированию.

В лабораторной работе №3 была создана основа приложения для голосования. 

В лабораторной работе №4 была проведена работа с формами: разработана система аутентификации и регистрации пользователей.

В практической работе была настроена авторизация OAuth2.0 в приложении.

В лабораторной работе №5 в приложение были добавлены микросервисы.
